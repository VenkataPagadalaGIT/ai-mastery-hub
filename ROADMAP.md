**How to Read Papers:**
1. **Title & Abstract:** Quick understanding
2. **Introduction & Conclusion:** Main contributions
3. **Figures & Tables:** Visual summary
4. **Methods:** Deep dive if interested
5. **Related Work:** Where it fits in the field

**Communities:**
- Papers with Code community
- Hugging Face community
- Reddit: r/MachineLearning, r/LanguageModels
- Discord: Fast.ai, Hugging Face, LangChain
- Local meetups and conferences

**Staying Updated:**
- ArXiv daily digest (e-mail subscription)

- Hugging Face blog (weekly)

**Trends to Watch:**
- Scaling laws and model size
- Multimodal models (text + image + audio)

- Open-source models closing the gap

**Pro Tips:**
- Don't read every paper; skim hundreds, read tens deeply
- Watch paper walkthroughs if you struggle

- Join research groups; contribute to open source
